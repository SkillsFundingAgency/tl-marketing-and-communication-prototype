Awarding organisation - Results

User journey for AO's to inform ESFA of each individual learners result for each registered component.

From: R&C Team

Published: 25 January 2022

Last updated: 25 January 2022

Journey overview

AO's inform ESFA of students results for each registered component.

Users add a result against components (Core and or Specialism(s)) for registered assessment entries.

For example if a student has taken an exam for their core in the summer of 2021, the AO must be able to update our service with the grade that the learner achieved for that exam.

End users

  • Results Editor (user role in the service)
  • Administrators

Pre requisites before adding results

  • AO must have reviewed the T Levels they offer within the R&C service.
  • AO must have reviewed which providers will offer their T Levels .
  • AO must have registered the learners from their providers.
  • Needs to have a current assessment entry to be registered against the learners record.
  • If a user is adding a result for a specialism. The learner must have a specialism registered and an assessment entry before a result can be added.

Current Functionality

  • Users are able to bulk upload results in a single CSV file.
  • Template provided to users for required data format - (Template is provided as a XLSX format not CSV).
  • Export results data via a CSV download.
  • Individually add or change a result for a learner using the learner's ULN.
