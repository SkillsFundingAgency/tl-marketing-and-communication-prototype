Awarding organisation - Assessments

User journey for AO's to inform ESFA which learners are taking an assessment (exam) in the current assessment window (Autumn/Summer).

From: R&C Team

Published: 25 January 2022

Last updated: 25 January 2022

Journey overview

AO's inform ESFA which students are taking an exam(assessment) in the current assessment window(Autumn/Summer). .

Information provided:

  • Core assessment entry
  • Occupational specialism assessment entry

End users

  • Registrations Editor (user role in the service)
  • Administrators

Pre requisites before adding assessments

  • AO must have reviewed the T Levels they offer within the R&C service.
  • AO must have reviewed which providers will offer their T Levels.
  • AO must have registered the learners from their providers.
  • Providers must have informed AOs which learners will be taking which exams when e.g. who will be taking core exams in Summer 2021.

Current Functionality

  • Users are able to bulk upload assessments in a single CSV file.
  • Template provided to users for required data format - (Template is provided as a XLSX format not CSV).
  • Export assessments data via a CSV download.
  • Individually add or remove an assessment entry for a learner using the learner's ULN.
