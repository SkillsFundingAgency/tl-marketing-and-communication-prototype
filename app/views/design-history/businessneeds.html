Business Needs

Content for BA to add and maintain

Appeals

Appeals description needed.

Appeals BN001

As the ESFA, we need to know when a grade is final, so that there is no uncertainty about the grade being changed and that it is ready for overall calculation.

11 February 2022

Appeals BN002

As the ESFA, we need to know when a grade is not final, so that we do not start overall calculations for the learner.

11 February 2022

Appeals BN003

As the ESFA, we need grades to be finalised as soon as they can be so that we are in the best position to calculate overall results on time.

11 February 2022

Appeals BN004

As the ESFA, we need to know when a grade has been changed, so that we know which grade to use for overall calculations.

11 February 2022

Appeals BN005

As the ESFA, we need to stop registrations being withdrawn if there is an outcome outstanding for a ROMM/appeal for a grade so that we have accurate information should a learner request an SOA.

11 February 2022

Data Export

Data export description needed.

Data Export BN006

As the ESFA, we need ability to export all core and specialism - registrations, assessments and results so that we can cross referencing data with provider data.

11 February 2022

Overall Results

Overall Results description needed.

Overall Results BN007

As the ESFA, we need to inform Providers what the overall results are, along with the breakdown (info going on the certificate - to allow providers to check what will be on the certificate) and to then generate certificates where appropriate. Applies to both Overall T Level certificate and Statement of Achievement (SOA).

11 February 2022

Overall Results BN008

As the ESFA, we need to restrict Providers and AOs ability to amend data once overall results has been calculated and certificate generated.

11 February 2022

Overall Results BN009

As the ESFA, we need automate generation of SOA for learners who did not receive an overall T Level but completed some of the components for a SOA to be available.

11 February 2022

Overall Results BN010

As the ESFA, we need to inform Providers when a learner is not going to receive a certificate so that they can investigate whether this is what the learner expects or meets what the Provider already knows.

11 February 2022

Overall Results BN011

As the ESFA, we need to inform Providers why a learner’s overall result has not been calculated so that they can investigate whether this is what the learner expects or meets what the Provider already knows.

11 February 2022

Overall Results BN012

As the ESFA, we need Industry Placement outcome to be up to date so that we know which learner’s results can be calculated.

11 February 2022

Overall Results BN013

As the ESFA, we need the ability to freeze all records for 2 weeks from COP 10 October to allow time between first and second batch of certificates being generated, relieving pressure of printing.

11 February 2022
